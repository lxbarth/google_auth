
Google Hybrid OpenID+OAuth
--------------------------

Authenticate user and authorize site to access user's resources in one step.

Setup
-----

1 Patch Drupal with
  http://drupal.org/files/issues/339600-19_google_openid_6.patch.
2 Install module.
3 Register your site with Google
  http://code.google.com/apis/accounts/docs/RegistrationForWebAppsAuto.html#new
4 Configure GData module with credentials obtained in 3.
